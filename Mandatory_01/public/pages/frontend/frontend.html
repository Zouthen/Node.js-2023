Fetch

Fetch is used to get data from the backend, and then use it in the frontend

Fetch is used like shown above, where you first write the url to the backend, and then you write the method, which is typycally GET."

then you write ".then", which is used to get the response from the backend, and then you write ".json()", which is used to convert the response to json, and then you write ".then" again, which is used to get the data from the response, and then you write the code you want to use the data for.
